About the project

What is the URL Challenge?

The URL Challenge is a DragonHack 2026 submission that explores how URL shorteners can do more than save characters. We built a fast, modern service that turns every link into a data point.

From real-time click analytics to custom slugs and QR codes, the challenge covers the full lifecycle of a shareable link — from creation to conversion.

Our goal: demonstrate clean API design, thoughtful UX, and production-ready infrastructure within a 24-hour window.

// POST /api/shorten
{
  "url": "https://example.com/very/long/path",
  "slug": "dh2026",
  "ttl": 86400
}

// 201 Created
{
  "short": "https://url.dh/dh2026",
  "clicks": 0,
  "qr": "https://url.dh/qr/dh2026"
}
What we built

Key features

Everything you'd expect from a modern link management platform — built in a single hackathon sprint.

⚡

Instant shortening

Paste any URL and get a short link in milliseconds — no account required.

📊

Click analytics

Track total clicks, referrer sources, device types, and geographic distribution in real time.

🎯

Custom slugs

Choose your own memorable slug or let the system generate a short, collision-free one.

🔒

Expiry & TTL

Set a time-to-live for links that should only be active during an event or campaign.

📱

QR code generation

Every short link comes with a downloadable QR code — perfect for posters and slide decks.

🌍

Open API

REST API with JSON responses and clear documentation so teams can integrate in minutes.
